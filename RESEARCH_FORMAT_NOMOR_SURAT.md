## Executive Summary

Format nomor surat resmi di Indonesia mengikuti standar BKN (Badan Kepegawaian Negara) dan Kepmendagri (Keputusan Menteri Dalam Negeri). Setiap instansi pemerintah, termasuk Dinas Pendidikan dan sekolah, memiliki format penomoran yang terstruktur dan dapat diverifikasi.

---

## 1. STRUKTUR DASAR NOMOR SURAT DINAS

### Komponen Utama:
```
[Nomor Urut] / [Kode Unit] / [Kode Klasifikasi] / [Bulan Romawi] / [Tahun]
```

### Contoh Format Umum:
```
001 / Disdik / III / 2026
```

**Arti:**
- `001` = Nomor urut surat ke-1 di bulan tersebut
- `Disdik` = Kode unit/instansi (Dinas Pendidikan)
- `III` = Bulan Maret (Romawi)
- `2026` = Tahun surat dikeluarkan

## 2. FORMAT NOMOR SURAT SEKOLAH (SD/SMP/SMA)

### Format Standar Kemendikbud:
```
[Nomor] / [Kode Sekolah] / [Kode Klasifikasi] / [Bulan] / [Tahun]
```

### Contoh:
```
001 / SDN.001.02.001 / 420 / III / 2026
```

**Arti:**
- `001` = Nomor urut
- `SDN.001.02.001` = Kode sekolah (SDN 001 Kabupaten 02 Kecamatan 001)
- `420` = Kode klasifikasi surat (contoh: 420 = Surat Tugas)
- `III` = Bulan Maret
- `2026` = Tahun

## 7. TATA CARA PENOMORAN SURAT DINAS

### Prinsip Dasar:
1. **Urutan berkelanjutan** - Nomor urut bertambah untuk setiap surat
2. **Per-bulan** - Nomor urut di-reset setiap bulan
3. **Per-tahun** - Tahun ditambahkan sebagai referensi
4. **Kode unit** - Menunjukkan instansi/organisasi pengirim
5. **Kode klasifikasi** - Menunjukkan jenis/jenis surat

### Aturan Penomoran:
```
- Nomor urut dimulai dari 001 setiap awal bulan
- Format: 001, 002, 003, dst
- Tidak ada skip nomor
- Nomor harus tercatat dalam register
```
