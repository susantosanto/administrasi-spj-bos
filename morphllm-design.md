# Design System: Morph - Inference Built for Coding Agents

> Extracted from [https://www.morphllm.com/](https://www.morphllm.com/) by brandmd

## 1. Visual Theme & Atmosphere

**Visual character:** Dark, high contrast; black background dominates with white text and green accents

**Density:** Moderate. The layout uses a varied spacing scale.

**Shape language:** Subtle rounding on interactive elements.

**Depth:** Uses 5 shadow styles for layering and elevation.

## 2. Color Palette & Roles

- **Dark gray** (`#27272A`): Divider / border (dominant)
- **White** (`#FAFAFA`): Light text (on dark) (dominant)
- **Near-transparent White** (`#FFFFFF1A`): Overlay / scrim (dominant)
- **Black** (`#0A0A0A`): Dark background / footer (dominant)
- **Green** (`#80EE64`): Link / accent text
- **Muted Green** (`#ACC3A7`): Muted text
- **Green** (`#99D52A`): Accent background
- **Green** (`#059345`): Accent background
- **Light Green** (`#BFF9B4`): Surface / card background
- **Near-transparent Vivid Orange** (`#F973161A`): Overlay / scrim

**Incidental (low usage, do not lead with these):** `#000000`

## 5. Layout Principles

**Spacing scale:** 4px, 6px, 8px, 10px, 12px, 16px, 20px, 24px

**Base unit:** 4px grid (values are mostly multiples of 4).

**Border radii:** 4px, 8px, 9px, 10px, 12px, 9999px (pill)

## 6. Guidelines

### Do
- Use 4px grid for all spacing
- Use `#80EE64` for primary actions and CTAs
- Stick to 4 font weights: 400, 500, 600, 700
- Use `GeistPixelCircle` as the primary typeface

### Don't
- Don't introduce colors outside the palette above
- Don't mix fonts beyond GeistPixelCircle and gogaFont
- Don't use inline styles when the design system covers the pattern
- Don't use border-radius values outside: 4px, 8px, 9px, 10px, 12px, 9999px (pill)
