# Dokumen Referensi

Tempat menyimpan dokumen referensi untuk diunduh oleh operator sekolah.

## File yang Diperlukan

| File | Keterangan |
|------|------------|
| `permendagri.pdf` | Peraturan Menteri Dalam Negeri terkait |
| `juknis-bosp.pdf` | Juknis BOSP dari Kemendikdasmen |
| `tka.pdf` | TKA dari Kemendikdasmen |
| `perbup.pdf` | Peraturan Bupati tentang transaksi tunai & non tunai |
| `ssh.pdf` | Standar Satuan Harga tahun terkini |
| `permendikbudristek-18.pdf` | Permendikbudristek nomor 18 tahun 2022 |

## Cara Menambahkan

1. Letakkan file PDF di folder ini
2. Pastikan nama file sesuai dengan yang tercantum di atas
3. Aplikasi akan otomatis mendeteksi file yang tersedia
