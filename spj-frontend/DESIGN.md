# Design System: CORETAX

> Extracted from [https://coretaxdjp.pajak.go.id/home-portal/id-ID/](https://coretaxdjp.pajak.go.id/home-portal/id-ID/) by brandmd

## 1. Visual Theme & Atmosphere

**Visual character:** Bright, high contrast; off-white background dominates with blue text

**Density:** Moderate. The layout uses a varied spacing scale.

**Shape language:** Rounded, friendly aesthetic with generous corner radii.

**Depth:** Uses 5 shadow styles for layering and elevation.

## 2. Color Palette & Roles

- **Off-white** (`#F8FAFC`): Page background (dominant)
- **Blue** (`#212C5F`): Primary text (dominant)
- **White** (`#FFFFFFCC`): Divider / border (dominant)
- **Near-transparent Dark Blue** (`#0F172A1F`): Divider / border (dominant)
- **Black** (`#000000`): Primary text
- **White** (`#FFFFFF`): Light text (on dark)

**Incidental (low usage, do not lead with these):** `#212C5F1A`, `#94A3B8`

## 3. Typography Rules

## 5. Layout Principles

**Spacing scale:** 1px, 6px, 8px, 10px, 11.2px, 22px, 24px, 56px

**Base unit:** 8px grid (values are mostly multiples of 8).

**Border radii:** 8px, 10px, 12px, 14px, 16px, 20px, 50%, 100px

## 6. Guidelines

### Do
- Use 8px grid for all spacing
- Use `#212C5F` for primary actions and CTAs
- Use `Plus Jakarta Sans` as the primary typeface

### Don't
- Don't introduce colors outside the palette above
- Don't mix fonts beyond Plus Jakarta Sans and Montserrat
- Don't use inline styles when the design system covers the pattern
- Don't use border-radius values outside: 8px, 10px, 12px, 14px, 16px, 20px, 50%, 100px
